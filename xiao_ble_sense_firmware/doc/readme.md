# pinout map

## Default (cannot change)
D4 - I2C SDA
D5 - I2C SCL
D6 - Serial Tx
D7 - Serial Rx
D11 - LED_RED
D12 - LED_BLUE
D13 - LED_GREEN

## Test_Rig Setup
D0 - white button GPIO pull-up
D1 - blue  button GPIO pull-up
D2 - motor encoder A
D3 - motor encoder B
D9 - motor PWM
D10 - motor direction
